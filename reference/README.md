# Reference Materials

This folder contains legacy source materials that are not part of the runtime userscript.

## `legacy-mispell/`

`legacy-mispell/` is a trimmed snapshot of the old original project files kept only as implementation reference.

Useful files:

- `legacy-mispell/src/bimbofy.js`: old transformation logic, useful for comparing rules.

The old `legacy-mispell/res/` data folder was removed from this repository. Runtime data should live in the um-typos JSON dictionaries under `data/<language>/`.
